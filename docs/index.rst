django-qshop documentation
==========================

django-qshop это интернет-магазин на джанго.

Он использует django-sitemenu для организации дерева сайта.

Обладает большой гибкостью. В отличие от стандарта в этой области: "засунуть в пакет всё что можно и потом этим не пользоваться", мы пытались сделать по принципу: "сделать все обязательное и добавить возможность легкого расширения"
